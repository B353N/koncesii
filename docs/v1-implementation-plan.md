# План за реализация на v1

Фазите са подредени така, че всяка завършва с работещ, демонстрируем резултат. Оценките са
за един опитен разработчик с AI агенти по конвенциите на репото.

## Фаза 0 — скелет (this repo) ✅→

- Монорепо скелет (pnpm + turbo), devcontainer, CI (gitleaks → lint → typecheck → test →
  docs integrity), AGENTS.md/CLAUDE.md, шаблони. Копира се от СИГМА и се преименува.
- **Изход:** зелено CI върху празните пакети.

## Фаза 1 — данните (bootstrap)

- `tools/harvest/` (вече написани): пълен НКР скрейп (1434+222), data.egov.bg extraction,
  качване в R2.
- `packages/ingest`: парсерите като тествани TS модули (портват логиката от Python) —
  windows-1251 TSV, Раздел I–XI парсер, хедър-евристиките за общинските CSV-та.
- `packages/db`: миграция `0000_init.sql` по [`core-scope.md`](core-scope.md), staging +
  domain + derived.
- Стар регистър (nkrold) обхождане.
- **Изход:** локален D1 с пълния единен модел; unify + dedup минават integrity gate.

## Фаза 2 — сайтът (read-only explorer)

- `apps/web` (RR v7 SSR по СИГМА шаблона): начало, /concessions (+детайл), /grantors,
  /companies (+cross-link към СИГМА по ЕИК), /search, CSV/JSON експорти, методология,
  достъпност, robots/sitemap.
- **Изход:** koncesii.com живее с пълните данни, без карта и без флагове.

## Фаза 3 — флаговете

- Derive стъпка + `/flags` страница по [`red-flags.md`](red-flags.md); флаг бадж на детайла
  на концесията с формулата и входните числа.
- **Изход:** сортируем списък на отклоненията с публична методология.

## Фаза 4 — картата

- Геокодиране на обектите (населено място + кад. идентификатори → координати; язовирите
  често имат кадастрален имот), MapLibre GL остров, клъстери по вид обект.
- **Изход:** /map с всички обекти; детайлът показва мини-карта.

## Фаза 5 — refresh автоматика

- `apps/etl` Worker: седмичен diff по НКР експорта + data.egov.bg update, integrity gate,
  blue/green D1 подмяна (по СИГМА ADR-0005).
- Решение за BG IP пътя (self-hosted runner във Варна или residential proxy) — ADR при
  избора.
- **Изход:** сайтът се обновява сам; badge „данни към ДД.ММ".

## v2 (след стабилен v1)

- Отчети за изпълнение (PDF extraction) → `ARREARS` флаг; МЕ плащания по находища; МТ
  плажове; свързани лица (от слоя на СИГМА); AI асистент по СИГМА спецификацията.

## Дефиниция на готово за v1

Публичен сайт с: пълния НКР (нов + стар) + общинските набори, единен модел с произход на
всяко число, флагове по публикуваната методология, карта, CSV/JSON експорти, автоматичен
седмичен refresh, зелено CI и документация без висящи препратки.
