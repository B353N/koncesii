# КОНЦЕСИИ

**Платформа за прозрачност на концесиите в България.** Публичен портал, който събира на едно място данните за всички концесии — язовири, морски плажове, добив на подземни богатства, обекти на публична собственост — от Националния концесионен регистър и отворените данни на държавата, свързва **концеденти → концесии → концесионери** и прави всяка сделка проследима до източника: кой обект, на кого е отдаден, за колко години, срещу какво възнаграждение и какво реално е платено.

Публичен портал: **[koncesii.com](https://koncesii.com)**

Концесията е най-дългосрочното разпореждане с публична собственост — язовир или плаж се отдава за 20–35 години, често срещу възнаграждение от няколкостотин лева годишно, а независим контрол се извършва върху ~5 договора годишно. КОНЦЕСИИ прави останалите ~1400+ видими: търсими, сравними, картографирани и сваляеми.

## Какво показва v1

Публичен, read-only справочник върху три същности и връзките между тях:

- **Концедент** — органът, който отдава (министър / кмет на община);
- **Концесионер** — компанията получател, ключ ЕИК;
- **Концесия** — партидата: обект, срок, възнаграждение, документи;

плюс **обектите на карта** (всяка концесия има локация) и **индикаторите за риск** с публична методология.

| Страница             | Route                  | На какъв въпрос отговаря                                        |
| -------------------- | ---------------------- | ---------------------------------------------------------------- |
| Начало               | `/`                    | Обобщение + най-дълги срокове, най-ниски възнаграждения          |
| Концесии             | `/concessions`         | Всички партиди, с филтри по вид, област, статус                  |
| Детайл на концесия   | `/concessions/[id]`    | Една сделка: обект, срок, пари, документи, флагове, източник     |
| Концеденти           | `/grantors`            | Кой орган какво е отдал                                          |
| Профил на концедент  | `/grantors/[id]`       | Една община/министерство: обекти, концесионери, суми             |
| Компании             | `/companies`           | Концесионерите; сортирани по брой/стойност                       |
| Профил на компания   | `/companies/[eik]`     | Един концесионер: какво държи, откога, срещу колко               |
| Карта                | `/map`                 | Всички обекти географски: язовири, плажове, находища             |
| Индикатори           | `/flags`               | Концесиите с червени флагове, по методологията                   |
| Търсене              | `/search`              | По обект, концесионер, концедент, номер на партида               |

Списъците имат CSV експорт, всяка концесия — JSON изглед. Методологията на всяко изчисление е публична, страниците са достъпни (WCAG) и не събират лични данни.

## Бърз старт

Хранилището е пригодено за разработка в Devcontainer (Docker + редактор с поддръжка).

```bash
pnpm setup    # еднократно: инсталиране + локален D1 + примерни данни
pnpm dev      # ежедневно: приложението + ETL worker-а паралелно
```

Първоначалното пълно извличане от регистрите (bootstrap) се прави с Python инструментите в [`tools/harvest/`](tools/harvest/) — от българско IP, защото регистрите режат datacenter адреси. Виж [`docs/etl.md`](docs/etl.md).
