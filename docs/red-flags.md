# Индикатори за риск — публична методология

Флаговете на КОНЦЕСИИ са **детерминистични, възпроизводими аритметични факти** върху
публикуваните данни. Флагът не е обвинение и не твърди нарушение — той казва „тази сделка
се отклонява от нормата по измерим начин и заслужава поглед". Всеки флаг на сайта линква
към този документ и към конкретните числа, от които е изчислен.

Принципи (задължителни за всеки нов флаг):

1. **Възпроизводимост** — всеки може да преизчисли флага от суровите данни с формулата тук.
2. **Неутрален език** — „годишното възнаграждение е 0,6% от стойността" е ок;
   „подозрителна сделка" не е.
3. **Праговете са явни** и се променят само през PR към този файл.
4. Флаг се изчислява в ETL derive стъпката, никога на живо в заявка.

## Флагове v1

| Код             | Условие                                                                   | Тежест |
| --------------- | ------------------------------------------------------------------------- | ------ |
| `LOW_PAYMENT`   | годишно възнаграждение / стойност на концесията < 1% (при налични двете)  | висока |
| `LONG_TERM`     | срок ≥ 300 месеца (25 г.); отделно ниво ≥ 420 месеца (35 г.)              | средна |
| `GRACE_PERIOD`  | гратисен период ≥ 24 месеца                                               | средна |
| `NO_INDEXATION` | липсва клауза за индексация при срок ≥ 120 месеца                         | средна |
| `SINGLE_BIDDER` | един участник в процедурата (където броят е публикуван)                   | висока |
| `YOUNG_COMPANY` | концесионерът е регистриран < 12 месеца преди решението (изисква ТР дата) | висока |
| `MISSING_MONEY` | нито еднократно, нито годишно възнаграждение е вписано                    | ниска  |
| `DATA_CONFLICT` | противоречиви стойности между източниците за едно поле                    | ниска  |

Пример (реална партида, използвана при валидацията на модела): концесия за строителство на
язовир със стойност 41 967,34 лв., срок 420 месеца и годишно възнаграждение 259,75 лв. дава
`LOW_PAYMENT` (0,62% годишно) + `LONG_TERM` (35 г.). Това е всичко, което флагът казва.

## Планирани (v2, изискват източника за изпълнение)

| Код             | Условие                                                         |
| --------------- | --------------------------------------------------------------- |
| `ARREARS`       | отчетено платено < договорено за периода                        |
| `NO_AUDIT`      | договор > 10 г. без нито една проверка (по докладите на АППК)   |
| `RELATED_PARTY` | свързаност концедент–концесионер (изисква слой „свързани лица") |

## Представяне

Всяка концесия показва флаговете си с код, изречение на неутрален език, формулата и входните
числа. Страницата `/flags` е сортируема по брой/тежест. CSV експортът включва флаговете, за
да могат журналисти да работят върху тях директно.

Промени по методологията минават като PR към този файл + миграция на derive стъпката в един
и същи changeset, за да не се разминават публикуваната методология и изчисленото.
