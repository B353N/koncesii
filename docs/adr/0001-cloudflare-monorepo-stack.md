# ADR-0001 — Монорепо на pnpm + turbo върху Cloudflare

- **Статус:** Прието
- **Дата:** 2026-07-10
- **Обхват:** цялото хранилище

## Контекст

КОНЦЕСИИ е read-only платформа за прозрачност върху държавен регистър: малък обем данни
(хиляди партиди), публичен трафик с пикове около публикации, нулев write път от потребители.
Екипът работи с AI-агентни работни потоци и иска ниска оперативна цена, edge кеширане и
възпроизводими деплойменти. Алтернативата — класически стек (напр. Laravel + Postgres, който
екипът също владее) — би дала позната среда, но по-висока оперативна цена и повече
инфраструктура за поддръжка при толкова малък обем данни.

## Решение

pnpm + turbo монорепо; Cloudflare Workers (React Router v7 SSR) + D1 + R2 + Queues/KV;
trunk-based разработка с conventional commits; AGENTS.md/CLAUDE.md механизъм за конвенциите;
CI верига gitleaks → lint → typecheck → test → docs integrity; ADR практика за решенията;
issue/PR шаблони. Отклонения се записват като нови ADR-и.

## Последствия

- Плюс: serverless без оперативна поддръжка; D1 + R2 покриват целия обем данни на практика
  безплатно; edge кешът поема пиковете; едно хранилище с единни конвенции за хора и агенти.
- Минус: зависимост от Cloudflare екосистемата; D1 лимитите важат (но при този обем не са
  фактор); извличането от регистрите изисква BG IP и живее извън Workers runtime-а.
