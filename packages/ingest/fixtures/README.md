# Fixtures за ingest тестовете

Малки извадки в точния формат на суровите източници: НКР TSV експорт
(windows-1251), HTML на партида и обявление (Раздел I–XI), data.egov.bg
`getResourceData` отговор.

Текущите файлове са **съставени по реалната структура** (проверена на живо,
docs/etl.md), но с примерни стойности. При първия локален harvest run се
заменят/допълват с реални извадки, за да се затвърдят парсерите срещу
действителните данни.

Правила: пълните сурови данни живеят в snapshot хранилището на сървъра и
**никога в git** — тук влизат само минималните извадки, нужни на тестовете.
Директорията е изключена от Prettier: фикстурите се пазят байт-точни, както
идват от източника.
