# Fixtures за ingest тестовете

Малки, реални извадки от суровите формати (НКР TSV експорт, HTML на обявление,
общински CSV от data.egov.bg), предоставени от локален harvest run. Пълните сурови
данни живеят в snapshot хранилището на сървъра и **никога в git** — тук влизат само минималните извадки,
нужни на тестовете.
